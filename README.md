# CJ Freshway Center App

React + TypeScript + Vite 기반의 CJ Freshway 센터 App

## 시작하기

### 개발 환경

```bash
# 의존성 설치
npm install
# 또는
yarn install
# 또는
pnpm install

# 개발 서버 실행
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
```

### 프로덕션 빌드

```bash
# 프로덕션 빌드
npm run build
# 또는
yarn build
# 또는
pnpm build
```

**빌드가 완료되면 `dist` 폴더가 생성됩니다. 이 폴더에는 운영 환경에 배포할 수 있는 모든 파일이 포함되어 있습니다.**

### 프로덕션 빌드 미리보기

```bash
npm run preview
# 또는
yarn preview
# 또는
pnpm preview
```

## 사용 가능한 스크립트

- `npm run dev` / `yarn dev` / `pnpm dev` - 개발 서버 실행
- `npm run build` / `yarn build` / `pnpm build` - TypeScript 컴파일 및 프로덕션 빌드 (dist 폴더 생성)
- `npm run preview` / `yarn preview` / `pnpm preview` - 프로덕션 빌드 로컬 미리보기
- `npm run lint` / `yarn lint` / `pnpm lint` - ESLint 실행
- `npm run lint:fix` / `yarn lint:fix` / `pnpm lint:fix` - ESLint 자동 수정
